# bvetra.by
Сайт компании Быстрее Ветра

# Run Locally
Clone the project

```bash
  git clone <repository-link>
```

```bash
  cd portfolio-website
  npm install
  npm run dev
```
# Tech Stack
- React
- TypeScript
- Vite
- React Router

# Features
- Адаптивный дизайн
- Многоязычность (Русский, Английский)
- Форма обратной связи
- SPA (Single Page Application) с использованием React Router
